Linux/Unix提升工作效率的几个工具

Linux
Linux, tools, zsh

下面是几个linux下挺很有用的玩意，可以大幅提高操作效率。包括好用的zsh，打开终端的工具，linux/unix文件转化工具等。

##终极shell:OH-MY-ZSH(吐血推荐)
mac和ubuntu默认的shell是bash，最近从网上看到了oh-my-zsh，不仅兼容bash，而且功能更强大，比如更加智能的模糊搜索，git项目高亮显示等，还可以自己配置主题，果断放弃bash。

mac安装请看着一篇文章:终极 Shell——ZSH

ubuntu安装:
安装zsh、git和wget：

sudo apt-get install zsh git wget

获取并自动安装oh-my-zsh：

替换bash为zsh：

chsh -s /bin/zsh

最后重启:

sudo reboot

重启后，打开terminal就可以看见新的shell了。
在用户主文件夹会生成一个.zshrc隐藏文件，可以添加个性化设置。比如我把python命令起了个别名alias py=”python”。只要输入py就可以用python解释器了。

##如何在文件浏览器中打开终端Terminal
这个还是挺常用的，有时候在文件浏览器、finder里面浏览文件想要打开终端时，打开终端还要输入文件路径，非常麻烦，解决方式：

mac：搜索这个工具go2shell然后安装，然后finder里就会看到这个>_<，点击就可以打开终端并且自动切换到当前文件路径。

ubuntu:
1）把终端加到右键菜单:
sudo apt-get install nautilus-open-terminal

3）重新加载文件夹:
nautilus -q

这样直接右键里边就有打开终端。

##如何在终端打开各种类型的文件
在mac的终端下用命令open可以调用相应的软件打开各种文件，比如我最常用的几个：

open .
调用finder打开当前路径（linux/unix中.表示当前文件夹）

open http://ningning.today
调用chrome打开网站等，比如上面我在.zshrc中用alias起了个别名，
alias blog="open http://ningning.today"
我在terminal中输入blog就可以打开我的博客了。

而在ubuntu下对应的命令是：
xdg-open,当然这个命令有点长，我在.zshrc中又添加一行alias xo=’xdg-open’
然后
xo t.txt #调用gedit打开t.txt

xo t.doc #调用office打开doc文件

xo . #和mac终端下执行open .一样的效果，用文件浏览器打开当前文件夹

##unix/linux和windows文件转换工具
由于不同操作系统的换行有差异，不同系统间传递文件可能会有些问题。安装俩工具就行，用法很简单，直接unix2dos后边跟上要转换的文件就行了。也可以用vim搞。
unix2dos
dos2unix
mac:
brew install unix2dos
ubuntu:
sudo apt-get install unix2dos
妈妈再也不用担心我在不同系统之间拷贝文件了（＾∇＾）
